# AnaliseExploratoria
Um commit é como salvar uma versão do seu trabalho no Git.
Quando faço um commit, eu registro as mudanças que fiz nos arquivos (por exemplo, correções, adições ou exclusões) e deixo uma mensagem explicando o que foi alterado.

É uma forma de guardar um ponto na história do projeto, para que depois eu possa ver o que foi feito, por quem e até voltar a versões anteriores, se precisar.
